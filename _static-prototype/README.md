# Portfolio Signal

Base de portafolio web con seccion de blog/newsletter, estilo cyberpunk corporativo y referencias visuales tipo YoRHa.

## Abrir

Con Laragon activo:

```text
http://localhost/portfolio-signal/
```

Tambien puede abrirse directamente desde:

```text
C:\laragon\www\portfolio-signal\index.html
```

## Editar contenido

- `index.html`: textos, secciones, entradas del blog y email de contacto.
- `styles.css`: paleta, layout, responsive y sistema visual.
- `script.js`: animacion de lineas y captura local del formulario de newsletter.
